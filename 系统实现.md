### 5.1.2 登录模块关键代码

本段代码创建了一个登录请求的处理逻辑，通过 Spring Boot 的 Controller 接收前端传递的登录表单数据。使用 JPA 查询数据库，查找与表单中提供的用户账号匹配的用户，以保证登录用户为表单中已存在的用户。如果找到了用户，并且用户提供的密码与数据库中存储的密码匹配则通过密码校验，执行登录操作并生成 JWT Token。最后的 switch 语句为重定向，根据用户的角色类型，将用户重定向到不同的页面，具体代码如图 5-2。

```java
@PostMapping("/login")
public ResponseMessage<Map<String, Object>> login(@RequestBody Map<String, Object> request) {
    String userAccount = String.valueOf(request.get("userAccount")).trim();
    String password = String.valueOf(request.get("password")).trim();
    
    // 验证用户账号是否存在
    User user = userRepository.findByUserAccount(userAccount);
    if (user == null) {
        return ResponseMessage.error("用户名或密码错误");
    }
    
    // 验证密码
    if (!passwordEncoder.matches(password, user.getPassword())) {
        return ResponseMessage.error("用户名或密码错误");
    }
    
    // 根据用户角色跳转到不同页面
    String redirectUrl;
    switch (user.getRole()) {
        case "学生":
        case "教师":
            redirectUrl = "/student/dashboard";
            break;
        case "实验室管理员":
            redirectUrl = "/lab/admin/dashboard";
            break;
        case "系统管理员":
            redirectUrl = "/system/admin/dashboard";
            break;
        default:
            redirectUrl = "/dashboard";
    }
    
    Map<String, Object> data = new HashMap<>();
    data.put("token", jwtUtil.generateToken(user));
    data.put("user", convertToUserDto(user));
    data.put("redirectUrl", redirectUrl);
    
    return ResponseMessage.success(data);
}
```

### 5.2.2 实验室管理关键代码

本段代码实现了实验室的分页查询功能，通过 Spring Data JPA 的 Pageable 接口实现分页逻辑。支持按实验室名称、分类ID、状态等条件进行动态筛选，使用 Specification 动态构建查询条件，提高了查询的灵活性。查询结果包含实验室的详细信息以及关联的分类和管理员信息，具体代码如图 5-4。

```java
@GetMapping
public ResponseMessage<Map<String, Object>> listLabs(
        @RequestParam(value = "page", required = false, defaultValue = "1") int page,
        @RequestParam(value = "size", required = false, defaultValue = "10") int size,
        @RequestParam(value = "name", required = false) String name,
        @RequestParam(value = "categoryId", required = false) Long categoryId,
        @RequestParam(value = "status", required = false) Integer status) {
    
    PageRequest pageable = PageRequest.of(page - 1, size, Sort.by(Sort.Direction.DESC, "id"));
    
    Page<Lab> pageResult = labRepository.findAll((root, query, criteriaBuilder) -> {
        List<Predicate> predicates = new ArrayList<>();
        
        if (name != null && !name.trim().isEmpty()) {
            predicates.add(criteriaBuilder.like(root.get("name"), "%" + name + "%"));
        }
        if (categoryId != null) {
            predicates.add(criteriaBuilder.equal(root.get("category").get("id"), categoryId));
        }
        if (status != null) {
            predicates.add(criteriaBuilder.equal(root.get("status"), status));
        }
        
        return criteriaBuilder.and(predicates.toArray(new Predicate[0]));
    }, pageable);
    
    Map<String, Object> data = new HashMap<>();
    data.put("records", pageResult.getContent().stream().map(this::convertToLabDto).toList());
    data.put("total", pageResult.getTotalElements());
    
    return ResponseMessage.success(data);
}
```

### 5.3.2 预约管理关键代码

本段代码实现了预约申请的创建功能，包含时间冲突检测和预约单号生成逻辑。通过查询数据库检查同一实验室在同一时间段是否已有预约，确保预约时间的唯一性。使用 UUID 生成唯一的预约单号，便于后续的预约管理和查询。预约创建后状态默认为待审核，等待管理员处理，具体代码如图 5-6。

```java
@PostMapping
public ResponseMessage<Map<String, Object>> createReservation(@RequestBody Map<String, Object> request) {
    Long labId = Long.valueOf(String.valueOf(request.get("labId")));
    Integer userId = Integer.valueOf(String.valueOf(request.get("userId")));
    LocalDateTime startTime = LocalDateTime.parse(String.valueOf(request.get("startTime")));
    LocalDateTime endTime = LocalDateTime.parse(String.valueOf(request.get("endTime")));
    
    // 检查时间冲突
    List<Reservation> conflicts = reservationRepository.findConflictingReservations(
        labId, startTime, endTime);
    if (!conflicts.isEmpty()) {
        return ResponseMessage.error("该时间段已被预约");
    }
    
    // 创建预约记录
    Reservation reservation = new Reservation();
    reservation.setOrderNo(UUID.randomUUID().toString().replace("-", ""));
    reservation.setUser(userRepository.findById(userId).orElseThrow());
    reservation.setLab(labRepository.findById(labId).orElseThrow());
    reservation.setStartTime(startTime);
    reservation.setEndTime(endTime);
    reservation.setPurpose(String.valueOf(request.get("purpose")));
    reservation.setStatus(0); // 待审核
    reservation.setUseStatus(0); // 待使用
    reservation.setCreatedTime(LocalDateTime.now());
    
    Reservation saved = reservationRepository.save(reservation);
    return ResponseMessage.success(convertToReservationDto(saved));
}
```

```java
@PostMapping
public ResponseMessage<Map<String, Object>> createRepair(@RequestBody Map<String, Object> request) {
    Repair repair = new Repair();
    repair.setUser(userRepository.findById(Integer.valueOf(String.valueOf(request.get("userId")))).orElseThrow());
    repair.setLab(labRepository.findById(Long.valueOf(String.valueOf(request.get("labId")))).orElseThrow());
    
    // 关联预约记录（可选）
    Object reservationIdObj = request.get("reservationId");
    if (reservationIdObj != null) {
        reservationRepository.findById(Long.valueOf(String.valueOf(reservationIdObj)))
            .ifPresent(repair::setReservation);
    }
    
    repair.setTitle(String.valueOf(request.get("title")));
    repair.setDescription(String.valueOf(request.get("description")));
    repair.setStatus(0); // 待处理
    repair.setCreatedAt(LocalDateTime.now());
    
    Repair saved = repairRepository.save(repair);
    
    // 通知实验室管理员
    notificationService.notifyLabAdmin(repair.getLab().getAdmin(), "新的报修申请", repair.getTitle());
    
    return ResponseMessage.success(convertToRepairDto(saved));
}
```

```java
@PostMapping
public ResponseMessage<Map<String, Object>> createMaintenance(@RequestBody Map<String, Object> request) {
    // 验证必填字段
    Object labIdObj = request.get("labId");
    Object contentObj = request.get("content");
    Object resultObj = request.get("result");
    Object handlerObj = request.get("handler");
    Object handlerPhoneObj = request.get("handlerPhone");
    
    if (labIdObj == null || contentObj == null || resultObj == null || 
        handlerObj == null || handlerPhoneObj == null) {
        return ResponseMessage.error("必填字段不能为空");
    }
    
    Maintenance maintenance = new Maintenance();
    maintenance.setLab(labRepository.findById(Long.valueOf(String.valueOf(labIdObj))).orElseThrow());
    maintenance.setContent(String.valueOf(contentObj));
    maintenance.setResult(String.valueOf(resultObj));
    maintenance.setMaintenanceUnit(String.valueOf(request.get("maintenanceUnit")));
    maintenance.setHandler(String.valueOf(handlerObj));
    maintenance.setHandlerPhone(String.valueOf(handlerPhoneObj));
    
    // 处理检修时间
    Object maintenanceTimeObj = request.get("maintenanceTime");
    if (maintenanceTimeObj != null) {
        maintenance.setMaintenanceTime(LocalDateTime.parse(String.valueOf(maintenanceTimeObj)));
    } else {
        maintenance.setMaintenanceTime(LocalDateTime.now());
    }
    
    maintenance.setCreatedTime(LocalDateTime.now());
    maintenance.setStatus(0); // 处理中
    
    Maintenance saved = maintenanceRepository.save(maintenance);
    return ResponseMessage.success(convertToMaintenanceDto(saved));
}
```

```java
    announcement.setCreatedAt(LocalDateTime.now());

    return ResponseMessage.success(convertToAnnouncementDto(saved));
```

### 5.7.2 权限控制关键代码

本段代码实现了基于角色的数据权限控制，通过 JWT Token 验证用户身份，并根据用户角色过滤可访问的数据。学生和教师只能查看自己的预约和报修记录，实验室管理员只能管理所辖实验室的相关数据，系统管理员具有全局访问权限。这种设计确保了数据的安全性和隐私保护，具体代码如图 5-14。

```java
@GetMapping
@PreAuthorize("hasAnyRole('学生', '教师', '实验室管理员', '系统管理员')")
public ResponseMessage<Map<String, Object>> listMaintenances(
        @RequestParam(value = "userId") Integer userId,
        @RequestParam(value = "role") String role,
        Pageable pageable) {
    
    Page<Maintenance> pageResult;
    
    switch (role) {
        case "学生":
        case "教师":
            // 只能查看自己报修的检修记录
            pageResult = maintenanceRepository.findByLabAdminId(userId, pageable);
            break;
        case "实验室管理员":
            // 只能查看所辖实验室的检修记录
            pageResult = maintenanceRepository.findByLabAdminId(userId, pageable);
            break;
        case "系统管理员":
            // 可以查看所有检修记录
            pageResult = maintenanceRepository.findAll(pageable);
            break;
        default:
            pageResult = Page.empty();
    }
    
    return ResponseMessage.success(convertToPageData(pageResult));
}
```
