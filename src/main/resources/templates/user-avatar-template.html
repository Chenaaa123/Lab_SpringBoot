Welcome to Lab Management System

This is a comprehensive laboratory management system that allows users to book labs, report issues, and track maintenance records. Your avatar is displayed in the top-right corner.

Avatar Features

Current Features:

  • Automatic avatar display from user profile
  • Support for Base64 image data
  • Support for external image URLs
  • Avatar upload functionality
  • Responsive design for mobile devices
  • Dark mode support

How to Update Avatar

1. Click on your avatar in the top-right corner
2. Click "Upload Avatar" button
3. Select an image file (max 5MB)
4. Wait for upload to complete
5. Your new avatar will appear automatically
